## 🚀 如何测试

### 1. 启动服务器
```bash
npm run dev
```

### 2. 访问页面
打开浏览器访问: http://localhost:9002/tiktok

### 3. 测试流程

#### 3.1 浏览创作
- 上下滑动浏览创作
- 点击右侧按钮（点赞、评论、收藏、分享）

#### 3.2 登录/注册
- 点击底部"我的"标签
- 点击"登录"按钮
- 输入邮箱和密码
- 点击"登录"或"注册"

#### 3.3 创作图案
- 点击底部"+"按钮
- 输入创意描述
- 选择艺术风格
- 点击"生成图案"

#### 3.4 生成商品
- 在首页点击创作
- 选择商品品类
- 等待生成
- 查看商品详情

#### 3.5 购买商品
- 在商品详情页选择尺码
- 点击"立即购买"
- 填写收货信息
- 填写支付信息
- 确认订单

#### 3.6 查看订单
- 进入个人中心
- 点击"订单"标签
- 点击订单查看详情

#### 3.7 发现页面
- 点击底部"发现"标签
- 浏览分类和标签
- 搜索创作

#### 3.8 消息页面
- 点击底部"消息"标签
- 查看通知
- 过滤通知类型

#### 3.9 设置
- 进入个人中心
- 点击设置图标
- 查看设置选项
- 退出登录
