# **App Name**: T-Shirt AI

## Core Features:

- Inspiration Upload: Users can upload an image to serve as inspiration for the t-shirt design.
- AI Pattern Generation: Generates unique t-shirt patterns based on user's text prompts and inspiration images. This feature utilizes a tool to decide whether to incorporate inspiration images into its generation.
- Style Selection: Allows users to choose from various artistic styles to apply to the generated pattern.
- T-Shirt Color Selection: Enables users to select the base color of the t-shirt.
- AI Model Mockup: Generates a realistic mockup of the selected pattern printed on a model wearing a t-shirt. Uses AI to modify the model to look similar to persons or characters in the inspiration image.
- Size and Quantity: Allows users to select the size and quantity of the t-shirt they wish to order.
- Order placement: Collects shipping and payment information to complete the order.

## Style Guidelines:

- Primary color: A vibrant cyan (#00BCD4) to reflect creativity and technology.
- Background color: A dark gray (#303030) to provide contrast and a modern feel.
- Accent color: A deep indigo (#3F51B5) to complement the primary color and add depth.
- Body and headline font: 'PT Sans' (sans-serif) to provide a modern and friendly aesthetic. 
- Code font: 'Source Code Pro' (monospace) for any code snippets that may appear in the app.
- Use clean, modern icons from Lucide-React to represent different functionalities. Icons should be cyan.
- Incorporate subtle animations and transitions to enhance user experience, such as fading effects and scaling transitions. Specific animations included are fadeIn, fadeInUp, scaleIn and popIn.
