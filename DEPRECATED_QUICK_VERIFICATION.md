## 🚨 常见问题检查

### 问题 1: 页面空白
**检查**:
- 浏览器控制台是否有错误
- 网络请求是否成功
- Firebase 配置是否正确

**解决**:
```bash
# 重启服务器
npm run dev
```

### 问题 2: 无法登录
**检查**:
- Firebase Console 是否启用匿名登录
- 授权域名是否包含 localhost
- 浏览器控制台的错误信息

**解决**:
访问诊断工具：http://localhost:9002/test-firebase.html

### 问题 3: 无法生成创意
**检查**:
- Gemini API Key 是否配置
- 用户是否已登录
- 网络连接是否正常

**解决**:
检查 `.env.local` 文件中的 `GEMINI_API_KEY`

### 问题 4: 点击无反应
**检查**:
- 浏览器控制台是否有 JavaScript 错误
- 按钮是否被禁用
- 是否在加载中

**解决**:
刷新页面重试
